01工具简介 Overview

ocgt-monitor 是一个独立的 OpenCode Go 套餐额度监控工具。它直接调用 OpenCode Go 的 RPC 接口查询额度，以及 DeepSeek 官方 API 查询余额。大部分功能与 ocgt 无关—只有历史消耗查询需要读取 ocgt 产生的 JSONL 日志文件。Most features work independently of ocgt; only the history/trend feature reads ocgt log files.

核心特性 Key Features
• 三种使用模式：CLI 终端 · 网页面板 · 桌面侧边栏
• 多账户管理，配置文件独立存储
• 双主题：亮色「灵动卡片」与暗色「深色专业」
• 侧边栏可拖拽定位、固定、自动隐藏
• 2 秒自动刷新，数据实时更新
• 开箱即用，仅依赖 WebView2 Runtime（Win11 自带）

02快速开始 Quick Start

1

下载程序 Download the program

从发布页下载 ocgt-monitor.exe，放到任意文件夹。Download the latest release from GitHub.

2

配置凭证 Set up credentials

打开终端（CMD），运行配置向导：Open a terminal and run the setup wizard.

ocgt-monitor config init

按提示输入以下信息。DeepSeek Key 可不填（余额功能不可用）。Follow the prompts. DeepSeek key is optional.

项目获取方式
Cookie登录 opencode.ai → F12 → Application → Cookies → 复制完整 Cookie 字符串
Workspace IDwrk_xxx 格式，从浏览器 URL 中获取
DeepSeek API Key（可选）platform.deepseek.com API Keys 页面
3

双击运行 Double-click to launch

直接双击 ocgt-monitor.exe，桌面侧边栏即刻启动，无需终端。Just double-click the exe — the sidebar appears on your right screen edge.

也可在终端运行 ocgt-monitor serve --sidebar 或 ocgt-monitor quota 等命令。CLI commands are also available from the terminal.

4

检查额度 Check your quota

也可直接在终端查看套餐额度：Alternatively, check quota directly in terminal.

ocgt-monitor quota
!
PowerShell 用户注意： 命令前需要加 .\ 前缀，例如 .\ocgt-monitor config init。PowerShell requires .\ prefix for local executables.

04桌面侧边栏 Desktop Sidebar

桌面侧边栏是 ocgt-monitor 的推荐使用方式。360×370px 半透明面板，固定在屏幕右侧边缘，鼠标移上时滑出，移开后自动隐藏。A frameless glass panel on the right edge of your screen. Auto-hides to a thin strip, slides out on hover.

启动方式：双击 ocgt-monitor.exe，或在终端运行 ocgt-monitor serve --sidebar。

数据面板说明 Data Panels

套餐额度

三条水平进度条：Rolling（滚动窗口）、Weekly（本周）、Monthly（本月，部分套餐无限制）。进度超过 80% 变橙色，超过 95% 变红色。

今日消耗

四张卡片：输入 Tokens、输出 Tokens、总计、请求次数。数字变化时有平滑滚动动画。

账户余额

显示可用余额、赠送金额、充值金额。未配置 DeepSeek Key 时隐藏。

模型消耗

按使用量排序，每行显示模型全称和 Token 总数。可切换 1日/7日/30日或自定义日期范围。

05配置管理 Configuration

ocgt-monitor 使用多 Profile 配置文件，每个 Profile 包含一套完整的凭证信息。

分享给他人
i
exe 本身不含任何配置，直接发送即可。配置存储在用户目录 C:\Users\<用户名>\.ocgt-monitor\config.json，每台机器独立。 接收方运行 config init 配置自己的凭证即可使用。

06网页面板 Web Dashboard

运行 serve 命令启动 HTTP 服务器，浏览器访问 http://127.0.0.1:8788 即可打开面板。

ocgt-monitor serve

也可通过 OCGT_PORT 环境变量自定义端口：

set OCGT_PORT=9090
ocgt-monitor serve

网页面板与侧边栏共享同一套 API 后端，数据一致。

07架构说明 Architecture

数据来源
OpenCode Go RPC (/_server)     →  quota    // 套餐额度
DeepSeek API (/user/balance)   →  balance  // 账户余额
ocgt JSONL 日志文件             →  history  // Token 消耗历史

CLI   →  ocgt-monitor quota|balance|history

08常见问题 FAQ

没配置凭证？

运行 config init 配置。

查 quota 报错？

Cookie 可能过期。重新登录 opencode.ai 复制 Cookie，运行 config init 更新。

history 无数据？

历史数据来自 ocgt 的 JSONL 日志文件，需要 ocgt 运行中才会产生日志。

侧边栏没反应？

需要 WebView2 Runtime。Windows 11 自带；Win10 请从微软官网安装。

需要安装 Go？

exe 可直接运行。只有需要重新编译时才需要 Go 1.22+。

如何更新配置？

重新运行 config init，旧值会自动作为默认值显示，直接回车保留。

面板能调透明度？

亮色/暗色主题已预设最佳透明度，可通过 CSS 变量自定义。

侧边栏位置重置？

拖拽到最上方或最下方松手即可归位。重启程序后位置自动恢复。
